Design Principles & Components

This website follows Atomic Design principles and utilizes Ant Design components for consistency and professionalism. Common elements like Navigation Bar and Footer maintain brand identity across all pages.

Atomic Design Hierarchy:
• Atoms: Basic UI elements (buttons, inputs, icons)
• Molecules: Simple component groups (search bars, menu items)
• Organisms: Complex UI sections (navigation, hero sections)
• Templates: Page layouts
• Pages: Final implementations with real content
